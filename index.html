Tableau Write-Back Extension

A Tableau Extension to allow send data to a REST endpoint

GitHub Download Manifest File Setup Extension

Description

The Tableau Write-back was implemented by Xpand IT and allows you to select data from your sheets, edit or add new information to it and persist it somewhere. This extension can be used, for instance to categorize (add more info to your categories) or add comments to your data.

As an easy way to plug and play the extension, we provide a script that you can use in your Google Spreadsheet to write back data to it. Nevertheless, the extension is flexible enough so you can plug it to your own service to write the results somewhere else.

Installing the Extension

Download and install the Extensions API version of Tableau Desktop from the Tableau Extensions API Developer Preview site. Under Resources, click Extensions API Software Downloads.
